# Airline Reservation System ✈️

An OOP-based Airline Reservation System with full GUI and database support.

## Features
- Passenger registration & login
- Interactive seat map (50 seats)
- Multiple payment methods (Credit Card, Cash, JazzCash)
- Booking & cancellation management
- Admin panel with CSV export
- SQLite database integration

## Tech Stack
Python 3 | OOP | Tkinter GUI | SQLite3 | ABC (Abstract Classes)

## How to Run
pip install tkinter
python airline_reservation.py
